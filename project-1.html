Project 1

Few-Shot Learning for Optic Nerve Sheath Image Segmentation Using Deep Learning and Clustering Techniques

Live Link

Project Overview

  • Research Project: Image Segmentation of the Optic Nerve Sheath (Few-Shots Learning Problem)
  • Employed advanced Deep Learning techniques, including ResNet and Unet, for precise image segmentation and feature extraction.
  • Utilized sophisticated clustering methods, such as HDBSCAN and Kmeans++, to enhance image classification accuracy.
  • Implemented Transfer Learning by training the decoder of a pretrained Unet on the project's dataset, significantly improving model performance.

Achievements

This project contributed to advancing image segmentation techniques for medical applications, specifically in the context of few-shots learning, which holds potential for more accurate and efficient diagnostic processes.
