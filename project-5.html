Detection of a Cup by a Ball Launching Robot Using Computer Vision Techniques

Live Link

Project Overview

  • Detection of a Cup by a Ball Launching Robot
  • Employed the Faster R-CNN model to detect the cup using a stereo camera (ZED 3).
  • Utilized image processing techniques for the extraction of 3D coordinates.

Achievements

This project contributed to the development of accurate object detection and localization techniques, crucial for the precise operation of the ball launching robot.
